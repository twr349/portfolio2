About Me

I’m a food enthusiast, world traveler, photographer, computer nerd, kitten foster, and a husband in addition to being a Web developer looking to expand my skills through creating new and interesting applications. My passion since very young always gravitated towards computers. I remember loading video games from floppy disks, using DOS, before I could even tie my shoes. Koios Note

A note taking application for the intellectual titan. Users can create different notebooks for each subject. Each notebook can have multiple topics, and each topic can have multiple notes. Koios Note allows the user to be able to take notes on their laptop and quick review them on their phone. Built on Ruby on Rails, as a progressive web application. With the help of service workers allows the user to add it to home screen with the appearance of a native application.

7 Days Todo

A self-destructing todo list. After user sign up, they can make tasks. The timer starts then. If the task is not completed with in 7 days it will self-destruct. Built on Ruby on Rails using Bootstrap. Has clean user interface also is mobile friendly. Allows the user to use their own avatar.
